FrontEnd

También conocido como lado del cliente, es la parte de un sistema o aplicación web con la que los usuarios interactúan directamente. El desarrollo frontend implica el uso de lenguajes de programación como HTML, CSS y JavaScript para crear páginas web interactivas y responsivas que se adapten a diferentes dispositivos y tamaños de pantalla.

BackEnd

También conocido como el lado del servidor, es la parte de un sistema informático que se encarga de procesar y gestionar los datos y la lógica de negocio. Esto se ocupa de tareas como el almacenamiento y recuperación de datos, la gestión de usuarios, la autenticación, la seguridad, el procesamiento de datos y cualquier otro cálculo necesario para que la aplicación funcione correctamente.

DataBase

Es una colección organizada de información estructurada que se almacena en formato digital. Una base de datos contiene tablas, que a su vez están compuestas por filas y columnas, donde se almacenan los datos de manera estructurada y coherente. Permite realizar consultas y manipular los datos de forma rápida y precisa.

Responsive Apps

Las aplicaciones responsivas son aquellas que se adaptan y ajustan automáticamente a diferentes tamaños de pantalla y dispositivos, proporcionando una experiencia de usuario consistente y fácil de usar. Esto permite que los usuarios accedan y utilicen la aplicación de manera efectiva sin importar el dispositivo que estén utilizando.
